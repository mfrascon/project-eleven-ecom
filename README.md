# Art Show Merch Store

An e-commerce web application built to sell postcards and stickers from our previous art show.  
This project focuses on a modern frontend experience using React and TypeScript, with Stripe handling secure product management and payments.

The goal of this project is to create a clean, fast, and scalable storefront that can grow alongside future art releases and events.
